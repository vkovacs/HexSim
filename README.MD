# Start Application
In the main folder of the application (HexSim) use
## Linux
~~~
./gradlew :gui:run
~~~

## Windows
~~~
gradlew :gui:run
~~~

# Configuration
## Config file location 
~~~
gui/src/main/java/hu/crs/hex/simulation/simulation/SimulationConfig.java
~~~
